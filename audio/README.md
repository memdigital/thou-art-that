## About the narration

Narrated by Nura, a voice persona we use across Marbl products. Nura is voiced using ElevenLabs (voice ID documented in our internal config; not tied to this repository). The written scripts are in `/narrations/`. The generated audio files sit alongside them as `.mp3` once produced.

The narrations are short distilled versions of the full written pieces. They are not substitutes for the written framework; they are a way in for listeners who prefer audio. If a topic matters to you, read the written version.

## Listening order

00 first if you are new to the piece. After that, the four principles stand alone - take them in the order that interests you.

If you only listen to one, pick 04 (Safety in Emergence). That is the principle that produced the rest of the framework.
